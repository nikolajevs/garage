# 🚨 Garage Alarm System — ESP32 + LD2410C + RC522

Умная охранная сигнализация для гаража на базе ESP32.

## Возможности

- Детекция присутствия человека с помощью радара **LD2410C** (mmWave)
- Управление через RFID-считыватель **RC522** (SPI)
- Задержки на выход (30 сек) и вход (15 сек)
- Фильтр ложных срабатываний
- Автоматическое отключение сирены через 3 минуты
- **Динамическое добавление карт через мастер-карту** — не нужно перепрошивать, чтобы добавить новую карту
- Все карты и мастер-карта хранятся в NVS (переживают перезагрузку и перепрошивку)

## Пины подключения

| Компонент       | Пин ESP32     |
|-----------------|---------------|
| LD2410C TX      | GPIO16 (RX2)  |
| LD2410C RX      | GPIO17 (TX2)  |
| RC522 SDA (SS)  | GPIO5         |
| RC522 RST       | GPIO4         |
| RC522 SCK       | GPIO18        |
| RC522 MOSI      | GPIO23        |
| RC522 MISO      | GPIO19        |
| RC522 3.3V      | 3.3V (⚠️ не 5V!) |
| Сирена/Реле     | GPIO32        |
| LED индикатор   | GPIO2         |

⚠️ RC522 питается строго от 3.3V — модуль обычно не имеет своего стабилизатора и не 5V-толерантен.
Сирену пришлось перенести с GPIO18 на GPIO32, так как GPIO18 занят аппаратным SPI (SCK) для RC522.

## Как работает

- **DISARMED** — Охрана выключена
- **EXIT_DELAY** — 30 сек на выход (медленное мигание LED)
- **ARMED** — Охрана активна (LED горит)
- **ENTRY_DELAY** — 15 сек на отключение (быстрое мигание)
- **ALARM** — Тревога! Сирена орёт

## Настройка

1. Загрузите прошивку и откройте Serial Monitor (115200 baud).
2. При первом запуске мастер-карта ещё не назначена — поднесите **любую** карту к RC522, она автоматически станет мастер-картой и сохранится в NVS.
3. Чтобы добавить обычную карту: поднесите **мастер-карту** (система должна быть снята с охраны, DISARMED) — LED начнёт часто мигать, это значит включён режим добавления. В течение 15 секунд поднесите новую карту — она сохранится в список разрешённых.
4. Повторное поднесение мастер-карты выключает режим добавления вручную (иначе он сам выключится через 15 сек).
5. Откалибруйте LD2410C через приложение HLKRadarTool.

### Serial-команды

| Команда | Действие |
|---|---|
| `list` | Показать мастер-карту и список обычных карт |
| `clear` | Удалить все обычные карты (мастер-карта не трогается) |
| `resetmaster` | Сбросить мастер-карту — следующая поднесённая карта станет новой мастер-картой |
