# PCB — несущая плата Garage Alarm

Проект KiCad (v7+, открывается и в KiCad 8/9) несущей платы под готовые модули.
Размер платы: **92 × 72 мм**, 2 слоя, 4 монтажных отверстия M3.

## Файлы

- `garage_carrier.kicad_pro` — файл проекта
- `garage_carrier.kicad_sch` — схема (связи заданы глобальными метками)
- `garage_carrier.kicad_pcb` — плата: все посадочные места расставлены, цепи назначены, земляной полигон на нижнем слое (B.Cu)
- `preview.svg` — картинка-превью компоновки
- `generate.py` — скрипт, которым сгенерирован проект (можно менять параметры и перегенерировать)

## Что нужно сделать в KiCad перед заказом

1. Открыть `garage_carrier.kicad_pcb`, нажать **B** — зальётся земляной полигон.
2. **Развести сигнальные дорожки по линиям ratsnest** (~15 цепей). Питание (+12V, +5V) — дорожки 1.0 мм, сигналы — 0.3 мм. Все компоненты выводные, поэтому дорожки можно вести по обоим слоям без переходных отверстий.
3. Запустить DRC, исправить замечания.
4. File → Fabrication Outputs → Gerbers + Drill → отправить на JLCPCB/PCBWay.

## Важное допущение

Гнёзда J1A/J1B рассчитаны на **30-контактный ESP32 DevKit (DOIT V1)** с межрядным
расстоянием **25.4 мм**. Если у вас 38-контактный DevKitC — измерьте свою плату и
поправьте координаты в `generate.py` (массив `FP`, строки J1A/J1B), затем перезапустите скрипт.
**Перед заказом обязательно приложите свой DevKit к распечатке платы в масштабе 1:1.**

## Подключение модулей

- **J2 (RC522)** — порядок пинов 1-в-1 как на модуле: SDA, SCK, MOSI, MISO, IRQ, GND, RST, 3.3V → прямой шлейф 8 pin.
- **J3 (LD2410C)** — порядок как на модуле: VCC(5V), GND, OUT, RX, TX.
- **J4** — вход питания 12V DC (клеммник).
- **J5** — сирена 12V: SIREN+ напрямую к +12V, SIREN− коммутируется MOSFET'ом (low-side).
- **J6** — геркон (задел на будущее, GPIO27).
- **U1** — посадочное место под DC-DC модуль 12V→5V (MP1584EN / Mini560). 5V идёт на VIN ESP32 и VCC радара.

## Схема ключа сирены

GPIO32 → R1 (100R) → затвор Q1; R2 (10k) затвор→GND (чтобы сирена молчала при
загрузке ESP32); сток Q1 → SIREN−; исток → GND; D1 катодом к +12V, анодом к
SIREN− — гасит выброс индуктивной нагрузки.
